# AnalyzeLLM4CppResult

This repository contains tools and coverage results for analyzing LLM output compared to C++ reference implementations.

Repository layout

- `ai_test_logs/` - collected LLM and reference test logs organized by project and component.
- `coverage_results/` - CSV coverage summaries generated by analysis scripts.
- `analyze_coverage.py` - analysis script used to aggregate coverage data.

Quick start

1. Inspect the coverage CSVs in `coverage_results/`.
2. To generate new coverage summaries, first unzip the relevant logs in `ai_test_logs/`.
3. Run the analysis script:
   python analyze_coverage.py
